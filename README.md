# Luany Framework

> Compiler-grade PHP MVC framework. Integrates luany/core and luany/lte.

## Installation

```bash
composer require luany/framework
```

Or start a new project from the official skeleton:

```bash
composer create-project luany/luany my-project
```

## What's included

- **luany/core** — Router, middleware pipeline, DI resolver hook
- **luany/lte** — AST-based template engine
- **vlucas/phpdotenv** — Environment variable loading

## Requirements

- PHP 8.1+
- Composer 2.0+
